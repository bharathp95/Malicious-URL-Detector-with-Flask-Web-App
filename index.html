Know what a malicious link looks like

☠️

Malicious Links — Red Flags

http://paypa1-secure-login.ru/verify https://amaz0n-account.info/reset?user=you http://bit.ly/3xFrEe-iPhone-2024
  • ▸ Misspelled brand names — paypa1, amaz0n, g00gle
  • ▸ Suspicious domains: .ru, .info, .xyz, .tk
  • ▸ Uses URL shorteners to hide the real destination
  • ▸ http:// instead of https:// — no encryption
  • ▸ Urgency words: "verify", "reset", "confirm now", "limited"
  • ▸ Random numbers or excessive hyphens in domain name
  • ▸ IP address used instead of a real domain — e.g. http://192.168.1.1/login
🛡️

Benign Links — Safe Signs

https://www.paypal.com/signin https://amazon.com/account/manage https://github.com/openai/gpt-4
  • ▸ Exact, correct brand spelling with a known domain
  • ▸ Trusted TLDs: .com, .org, .gov, .edu
  • ▸ Full visible URL — no shortener, no redirection
  • ▸ Always uses https:// with a padlock in your browser
  • ▸ Clean, readable path that makes logical sense
  • ▸ No excessive subdomains, hyphens, or random strings
  • ▸ Matches the context it was shared in — email, website, message
⚡ Quick Safety Rules
01

Hover before you click — check the URL shown in the status bar first.

02

Never trust a link sent in an unsolicited email or text message.

03

Expand short URLs using a preview tool before visiting them.

04

Always check for HTTPS and a padlock icon in the browser bar.
